# IP_Cam_Decoder
A demo application to decode H264 ip cam  images using qt and ffmpeg


# Compiling:

1) Clone this repository
2) cd IP_Cam_Decoder
3) mkdir build
4) cd build
5) cmake ..
6) make

# How can i use it?

1) Put the IP of your IP cam in the text box, like that: rtsp://ip:port
2) Press connect
3) To exit, press quit
